# 사용자 페르소나 - 테이블 오더 애플리케이션

---

## Persona 1: 고객 (Customer)

- **이름**: 김용훈
- **역할**: 매장 방문 고객
- **특징**:
  - 테이블에 앉아 태블릿으로 주문
  - 별도 로그인 없이 즉시 주문 가능
  - 메뉴 탐색, 장바구니 관리, 주문 생성, 주문 내역 확인
- **목표**: 빠르고 직관적으로 원하는 메뉴를 주문하고 주문 상태를 확인
- **불편함**: 직원을 기다리는 시간, 주문 실수, 주문 상태 불투명

---

## Persona 2: 매장 운영자 (Store Owner / Admin)

- **이름**: 허승룡
- **역할**: 매장 주인 또는 홀 매니저
- **특징**:
  - 관리자 화면에서 실시간 주문 모니터링
  - 메뉴 등록/수정/삭제 관리
  - 테이블 세션 관리 (이용 완료 처리)
  - JWT 기반 16시간 세션 로그인
- **목표**: 실시간으로 주문 현황을 파악하고 효율적으로 매장을 운영
- **불편함**: 주문 누락, 테이블 상태 파악 어려움, 메뉴 변경 번거로움
